# Oxygen Theme for Obsidian

A fork of [Minimal](https://github.com/kepano/obsidian-minimal) theme with enhanced customization features. Find detailed documentation at [minimal.guide](https://minimal.guide).

Oxygen is an [Obsidian](https://obsidian.md/) theme for desktop, mobile and tablet. Customize colors, fonts and more with the companion plugins [Oxygen Theme Settings](https://github.com/david/obsidian-oxygen-settings) and [Hider](https://github.com/kepano/obsidian-hider).
